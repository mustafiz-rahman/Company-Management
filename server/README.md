## App Name: Employee Managemen System
## Description 
    This is a service based app. Multiple company can use the app.
## Technologies to build the App
    DataBase : postgresql
    Backend: 
        framework: Nestjs
        Programming Language: Typescript

    Frontend:   
        framework: Reactjs,MUI
        Programming Language: Typescript 


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
